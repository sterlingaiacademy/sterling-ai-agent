How to get your credentials

Follow these step-by-step guides to collect every piece of information needed to activate Sterling AI Assistant. Each section covers one credential with exact navigation instructions.

1
WhatsApp Access Token
From Meta Developer Console
1
Go to developers.facebook.com and log in with your Facebook account
2
Click My Apps at the top → select your app (or create one if needed)
3
In the left sidebar, click WhatsApp → then click API Setup
4
Click the blue "Generate access token" button → select your WhatsApp Business account
5
Copy the long token that appears in the text box — it starts with EAA…
⚠ Tokens expire after ~24 hours in development mode. If your agent stops working, regenerate the token and update it in Setup → Step 2.
2
Phone Number ID
From Meta Developer Console
1
On the same API Setup page (WhatsApp → API Setup)
2
Under "Send and receive messages", find the From dropdown with your phone number
3
Below the dropdown you will see Phone Number ID: 106640715… — click the copy icon next to it
4
This is a numeric ID — it is NOT your actual phone number (e.g. +91 89219 25327)
💡 The Phone Number ID and Access Token are always on the same page. Bookmark it for easy access when tokens expire.
3
Google Sheet ID
From Google Sheets URL
1
Open Google Sheets and open the expense tracker spreadsheet
2
Look at the browser URL bar — it will look like: docs.google.com/spreadsheets/d/ABC123.../edit
3
Copy the part between /d/ and /edit — that is your Sheet ID
4
Make sure Row 1 has these headers: credit / debit · purpose · amount · balance
5
The Sheet must be owned by or shared with the Google account you connected in Step 1
4
Configure Meta Webhook
After setup is complete
1
In Meta Console → WhatsApp → Configuration
2
Under Webhook, click Edit
3
Set Callback URL to: your-server-url/webhook/whatsapp
4
Set Verify Token to the same passphrase you entered in Setup Step 2
5
Click Verify and Save → then subscribe to the messages field
💡 Using ngrok for local testing? Your webhook URL will be: https://xxxx.ngrok-free.app/webhook/whatsapp — update it every time ngrok restarts.
