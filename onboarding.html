Implementation Guide

1. Meta Business Setup

  1. Go to business.facebook.com/settings.
  2. Under Users, select System Users and create one.
  3. Generate a token checking whatsapp_business_messaging.

2. Google Sheets Structure

Ensure your tracking sheet has columns for: Date, Credit/Debit, Purpose, Amount, Balance.

3. Webhook Linking

In Meta API settings, set callback URL to your Render domain + /webhook/whatsapp.
